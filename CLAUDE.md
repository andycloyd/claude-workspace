# Claude Workspace

This is Andy Cloyd's Claude Code workspace — a central repo for scripts, tools, and automation built with Claude.

## Conventions
- Use Python 3 for scripts unless there's a reason not to
- Keep things simple and well-documented
- Use GitHub Actions for cloud automation

## Project Structure
- `scripts/` — Utility scripts and tools
- `.github/workflows/` — GitHub Actions for cloud-based Claude automation
